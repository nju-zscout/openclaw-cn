# `moltbot-cn pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
moltbot-cn pairing list whatsapp
moltbot-cn pairing approve whatsapp <code> --notify
```
